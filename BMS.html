遊玩說明

請依照遊玩步驟進行遊戲下載及安裝

請先下載以下所有軟體

RadminVPN LocaleEmulator 安裝插件

下載完成包含BMS186.7z 你會得到兩個資料夾 一個host檔案 一個install.exe 跟RadminVPN的安裝檔

資料夾1:冒险岛online186(遊戲客戶端)

資料夾2:Locale Emulator(改語言程式)

檔案:host(搭配install.exe使用)

檔案:install.exe(搭配host使用)

安裝檔:RadminVPN(雙擊安裝)

上述檔案都有之後 先安裝RadminVPN

第二步請用系統管理員身分 打開install.exe

它會自動幫你改寫系統host 以便順利登入遊戲

第三步請到Locale Emulator資料夾

以系統管理員身分打開LEInstaller.exe

選擇第二排 Install for all User (左邊那個)

第四步用系統管理員打開LEGUI.exe

把設定改為這樣 選擇另存為...

設定一個名稱 如圖 我用的是CN

再來就可以打開遊戲資料夾

滑到底選擇冒險島萬能登入器 右鍵選擇Locale Emulator

再選擇你剛剛的設定檔 例如:CN

如果你有在選單看到你的設定檔 也可以直接點 如圖

輸入Discord裡面RadminVPN的IP選擇連接服務器

連接成功會顯示服務器延遲 然後就可以點進入遊戲

進入遊戲點下去會要你選擇超級模式或是網卡模式

這裡如果你電腦配置很好 就點超級模式

如果你電腦或網路比較慢 選網卡模式

第一次登入會在登入那邊卡比較久是正常的

還有一定要用視窗模式 如果全螢幕玩

你只會看到盛大登入 那個是登不進遊戲的

以上 有問題可以在DC的問題集參考答案
